AI 核心术语

Core AI Terminology

一条视频无痛带你捋清楚这些全部的专业AI术语。今天咱们就用最通俗的语言，让你彻底搞懂AI的核心理念。

大语言模型

Large Language Model (LLM)

你每天在用的豆包、通义千问、Kimi等，在AI圈里更专业的称呼。它们是经过海量数据训练，能理解和生成人类语言的AI模型。

提示词

Prompt

你向AI发出的指令或问题。例如“帮我规划一趟去成都的3日旅行”。好的 prompt = 明确 + 有上下文。

Token

计费与计算单位

AI处理语言的最小单位，无论是中文、英文还是代码，都会被拆分成Token来计算工作量和费用。中文通常1-2个字为一个Token，但英文按词根切分，导致AI数不清单词里的字母数，比如 "strawberry"。

上下文能力

Context Window

AI记住并利用之前对话内容的能力。上下文越长，AI的记忆力越好，但计算量会呈平方级增长，成本也更高。

M

多模态

Multi-modality

AI处理不同信息类型（模态）的能力，如文字、图片、语音。核心技术是“模态对齐”，让AI在同一空间理解不同信号。

大模型幻觉

Hallucination

AI一本正经地编造看似合理但实则虚构信息的情况。这是因为它的目标是预测“最可能的下一个词”，而非说真话。
R

强化学习

Reinforcement Learning

通过奖励和惩罚机制训练AI。当AI回答得好时给予“奖励”，不好时给予“惩罚”，让它学会生成更好的内容。当奖励来自人类打分时，称为RLHF。

检索增强生成

Retrieval-Augmented Generation (RAG)

解决幻觉的另一种方法：让AI在回答前，先去搜索引擎或外部数据库查资料，再根据查到的资料生成答案，让回答有据可依。

1. 检索 (Retrieve)

从外部知识库查找相关信息。

2. 生成 (Generate)

结合检索到的信息生成答案。
